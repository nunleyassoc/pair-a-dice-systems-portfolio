# Dialog System

A lightweight dialog system used to display speech.

## Features
- Dialog anchored to world-space entities
- Central dialog manager for timing and cleanup
- Supports a custom multi-line dialog, pausing, and calling functions
- Designed to avoid pausing gameplay

Used for storytelling!
Creatures will make remarks like "blub blub" for a Pufferfish
