# <Pair A Dice> – Core Systems (Portfolio)

This repository contains a curated selection of core gameplay and
technical systems from my shipped Godot game (~20k lines of code total).

The full game, assets, and content are not included in this repository.
This code is shared solely for portfolio and evaluation purposes.

## What’s Included

- Enemy AI using a reusable state machine architecture
- Procedural spawning and world logic
- Combat resolution and hit detection systems
- Data-driven configuration using Godot resources
- Custom signal/event architecture for decoupled systems

Each script is documented with its purpose and design tradeoffs.

## What’s Not Included

- Game assets (models, audio, textures)
- Levels / scenes required to play the full game
- Steam build or release infrastructure

## About the Project

- Engine: Godot 4.2
- Scope: Solo-developed project
- Status: Released on Steam
