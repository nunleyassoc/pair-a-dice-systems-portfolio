# Crafting

A data-driven crafting system used to combine collected resources
into new items.

## Features
- Recipe-based crafting logic
- Clear separation between input validation and result creation
- Designed to be easily expandable with new recipes

This system emphasizes simplicity and extensibility over hard-coded logic.
